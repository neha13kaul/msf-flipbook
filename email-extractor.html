Create the Power Automate Flow

Build a manual-trigger flow that reads your inbox and populates the Excel table

▼

Create the flow

  1. Go to make.powerautomate.com and sign in with your Microsoft account.
  2. Click + Create → Instant cloud flow.
  3. Name it "Inbox to Excel", choose trigger Manually trigger a flow, click Create.

Action 1 — Get emails from Inbox

Click + New step, search for Office 365 Outlook, select action Get emails (V3). Configure:

SettingValue
FolderInbox
Fetch Only Unread MessagesNo
Include AttachmentsNo (faster)
Top500 — or however many you want (max 999)
Order ByDateTimeReceived desc
Search QueryLeave blank for all, or enter a date range like received>=01/01/2025

Action 2 — Loop over each email

Click + New step, search for Apply to each. In the Select an output from previous steps field, pick the dynamic content value value from the Get emails step.

Actions 3–11 — Compose each field (inside the loop)

Inside the Apply to each loop, add a Compose action for each field below. Name each Compose action using the column name (e.g. Compose – Sender Name) so it's easy to map later.

Sender Name
if(contains(items('Apply_to_each')?['From'], '<'), trim(substring(items('Apply_to_each')?['From'], 0, indexOf(items('Apply_to_each')?['From'], '<'))), items('Apply_to_each')?['From'])
Sender Email
if(contains(items('Apply_to_each')?['From'], '<'), substring(items('Apply_to_each')?['From'], add(indexOf(items('Apply_to_each')?['From'], '<'), 1), sub(sub(length(items('Apply_to_each')?['From']), 1), add(indexOf(items('Apply_to_each')?['From'], '<'), 1))), items('Apply_to_each')?['From'])
Date / Time (converted to your local timezone — change "Eastern Standard Time" if needed)
convertFromUtc(items('Apply_to_each')?['DateTimeReceived'], 'Eastern Standard Time', 'MM/dd/yyyy HH:mm')
Subject
items('Apply_to_each')?['Subject']
Company (extracts domain root — e.g. "microsoft" from @microsoft.com)
first(split( last(split( if(contains(items('Apply_to_each')?['From'], '<'), substring(items('Apply_to_each')?['From'], add(indexOf(items('Apply_to_each')?['From'], '<'), 1), sub(sub(length(items('Apply_to_each')?['From']), 1), add(indexOf(items('Apply_to_each')?['From'], '<'), 1))), items('Apply_to_each')?['From']), '@')), '.'))
Note: The Company expression returns the root domain name (e.g. "microsoft", "salesforce"). Personal email providers (gmail, yahoo, hotmail, outlook, icloud, live) will show those domain names — you can filter them out in Excel after export using a column filter.
Signature Text (approximation from the last 300 characters of the body preview)
if(greater(length(items('Apply_to_each')?['BodyPreview']), 100), substring(items('Apply_to_each')?['BodyPreview'], sub(length(items('Apply_to_each')?['BodyPreview']), min(300, length(items('Apply_to_each')?['BodyPreview']))), min(300, length(items('Apply_to_each')?['BodyPreview']))), '')
Note: Power Automate's body preview is limited to 255 characters. For true signature extraction from the full HTML body, add a separate Get email (V2) action inside the loop (using the message ID), then use items('Apply_to_each')?['Id'] as the Message ID. The full body content will be available as Body/Content. This doubles the API calls so it's slower.
Other Recipients (To + CC, excluding yourself)
concat( items('Apply_to_each')?['To'], if(empty(items('Apply_to_each')?['Cc']), '', concat('; ', items('Apply_to_each')?['Cc'])))
Has Attachments
if(items('Apply_to_each')?['HasAttachment'], 'Yes', 'No')
Email Preview (first 200 characters of body)
substring(items('Apply_to_each')?['BodyPreview'], 0, min(200, length(items('Apply_to_each')?['BodyPreview'])))

4

Run the Flow & Download Your Excel File

Trigger the flow, wait for it to complete, then download from OneDrive

▼
  1. On the flow detail page, click Run → Run flow → Done.
  2. Watch the run history at the bottom of the page. For 500 emails, the flow typically takes 3–8 minutes.
  3. Once the status shows Succeeded, go to OneDrive and open EmailExport.xlsx.
  4. Click Download to save it locally, or open it directly in Excel Online.
To re-run later: Clear the existing rows in the EmailData table first (select all data rows in Excel → Delete rows), then trigger the flow again. Or create a new copy of the template for each export.
Rate limits: Power Automate throttles Office 365 connector calls. If your flow fails mid-way with a "429 Too Many Requests" error, add a Delay action (1–2 seconds) inside the loop, or reduce the number of emails fetched. M365 Business plans have higher limits than personal plans.
